Installation
===================

``HBUtils`` is currently hosted on PyPI. It required python >= 3.6.

You can simply install HBUtils from PyPI with the following command:

.. code:: shell

    pip install hbutils

You can also install with the newest version through GitHub:

.. code:: shell

    pip install -U git+https://github.com/hansbug/hbutils.git@main

of ``hbutils`` should be shown.
